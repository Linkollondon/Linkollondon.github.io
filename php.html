    Применение

    1

    Cкриптовой язык, который очень активно применяется при разработке веб-приложений. Аббревиатура PHP означает “Hypertext Preprocessor (Препроцессор Гипертекста)".

    PHP — это препроцессор гипертекста (HTML).

    PHP — это серверный язык программирования.

    PHP — это скриптовый, интерпретируемый язык программирования.

    Основная задача PHP — это «оживление» HTML страниц.

    2

    Часто пользователи приходят на сайт за информацией, которая всё время меняется, и нужно отображать её актуальное состояние. Например:

    -показать курс валют

    -подсказать погоду на завтра

    Если использовать только HTML, то решить такие задачи не получится. Здесь-то и пригождается PHP.

    Он принимает входящий запрос от веб-сервера, выполняет сценарий и возвращает веб-серверу результат в виде готового HTML-код.

    3

    Сервер отправляет этот результат в браузер пользователю, который, в свою очередь, отображает её пользователю. После этого видно свежий курс валют, погоду, и что угодно ещё.

    РНР позволяет изменять веб-страницу на сервере непосредственно перед тем, как она будет отправлена браузеру.

    Когда браузер запрашивает страницу с адресом, который кончается на .php, веб-сервер обращается к PHP, и просит его выполнить сценарий из файла по этому адресу.

    PHP в масштабных проектах

    Основная сфера применения языка PHP — это веб, то есть сайты, которые мы каждый день посещаем через браузер компьютера или смартфона.

    Более известными работами на данный момент являются Yahoo, Flickr, Digg

    Электронная коммерция - популярные онлайн-платформы, такие, например, как OpenCart, Zen Cart, Magento, PrestaShop и Ubercart

    Создание CMS — одно из наиболее популярных использований PHP. На нем работают три ведущих движка для CMS: WordPress, Joomla и Drupal.
